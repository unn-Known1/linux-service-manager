# Secret Rotation Orchestrator

## Overview

Orchestrate secret rotation across systems: generate new secrets, update dependent services atomically, handle rollback, maintain audit trail.

## Problem Addressed

Rotating secrets (database passwords, API keys, certificates) is security best practice but manually risky and operationally complex.

## Why This Tool Exists

Existing solutions: HashiCorp Vault (rotation custom), AWS Secrets Manager (AWS only), CyberArk (enterprise)

**Gap:** Multi-cloud, open-source, GitOps-friendly approach with clear audit logs

## Installation

### Option 1: Direct Download
```bash
chmod +x secret-rotation-orchestrator.go
sudo cp secret-rotation-orchestrator.go /usr/local/bin/secret-rotation-orchestrator
```

### Option 2: Via Repository
This tool is part of the linux-service-manager toolkit. Simply run:
