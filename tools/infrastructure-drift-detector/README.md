# Infrastructure Drift Detector

## Overview

Continuous drift detection that compares live infrastructure against IaC definitions, alerts on discrepancies, and can auto-remediate.

## Problem Addressed

IaC tools (Terraform, CloudFormation) define desired state, but actual infrastructure can drift due to manual changes, emergencies, or bugs.

## Why This Tool Exists

Existing solutions: AWS Config, Azure Policy (cloud-specific), drift detection in commercial tools

**Gap:** Vendor-neutral, supports multiple IaC tools and cloud providers

## Installation

### Option 1: Direct Download
```bash
chmod +x infrastructure-drift-detector.py
sudo cp infrastructure-drift-detector.py /usr/local/bin/infrastructure-drift-detector
```

### Option 2: Via Repository
This tool is part of the linux-service-manager toolkit. Simply run:
```bash
./infrastructure-drift-detector.py --help
```
