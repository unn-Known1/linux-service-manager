# Incident Timeline Builder

## Overview

Automatically ingest data from monitoring, logging, chat (Slack/Discord), deployments, and build an interactive timeline with AI-assisted event correlation.

## Problem Addressed

During/after incidents, building a timeline from logs, metrics, deployments, and chat is manual and time-consuming.

## Why This Tool Exists

Existing solutions: PagerDuty Timeline, OpsGenie (basic), manual timelines in incidents

**Gap:** Open-source, integrations with popular tools (Grafana, Loki, Sentry, Slack), AI correlation

## Installation

### Option 1: Direct Download
```bash
chmod +x incident-timeline-builder.py
sudo cp incident-timeline-builder.py /usr/local/bin/incident-timeline-builder
```

### Option 2: Via Repository
This tool is part of the linux-service-manager toolkit. Simply run:
